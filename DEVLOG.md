# 辉辉提词器 开发日志

## v0.0.1 — 起步：从 Python 搬到手机上
决定把备播Agent从 Python 本地服务做成独立 Android App。之前电脑一关就不能用，主播在外面根本没法用。选了 Capacitor 方案，前端 JS 直接调 Kimi API，不需要后端服务器。

写了三个核心文件：db.js（IndexedDB 存储）、api.js（LLM 调用 + prompt 逻辑）、index.html（界面）。把 Python 版的 6 个 LLM 调用全部移植成 fetch。

第一次在手机上跑起来的时候，还挺激动的。

## v0.0.2 — 加提词器，直接翻车
想法是生成完文案直接送到提词器，打开相机录视频的时候也能看到。做成了 Android 系统级悬浮窗（SYSTEM_ALERT_WINDOW），写了原生 Java Service。

结果一启动就崩了。

## v0.0.3 — 修崩溃：Android 14 的坑
查了半天，Android 14（targetSdk 36）对前台服务要求变了，需要三个东西一起配：
1. FOREGROUND_SERVICE_SPECIAL_USE 权限
2. property 标签声明 subtype
3. startForeground 要传 ServiceInfo.FOREGROUND_SERVICE_TYPE_SPECIAL_USE

三个全漏了。补上之后终于不崩了。

## v0.0.4 — 透明度不够
提词器背景太黑了，录视频的时候完全看不到自己在画面里的位置。把背景透明度从 180 调到 60，加了文字阴影保证白字可读。

## v0.0.5 — 改名：备播Agent → 辉辉提词器
把 app 名字从「备播Agent」改成「辉辉提词器」。改了 capacitor.config.ts 和 strings.xml。注意只改 app 名，按钮上还是叫「提词器」。

## v0.0.6 — 提词器升级为第四个 Tab
原来只有三个 tab（生成、改写、参考），现在提词器独立成第四个 tab。底部导航栏四个图标。

## v0.0.7 — 速度按钮换中文
把提词器控制条的 🐢🐇 换成「慢」「快」两个字。录视频的时候看 emoji 不直观。

## v0.0.8 — 问题模式大改
重写了「问题」模式的 prompt。之前生成的问题太泛，不够具体。现在针对目标受众（大学生、职场困惑的人、想入行AI的人），问题分类型标签：【故事】【案例】【方法】【过程】【对比】【深度】。好问题的标准是主播听完脑子里马上有画面、有话想说。

## v0.0.9 — 收藏功能 + 右上角菜单
加了收藏功能，生成的文案可以点收藏。右上角加了 ☰ 菜单，下拉有「我的收藏」「设置」「使用指南」。IndexedDB 升级到 version 2，加了 favorites store。

## v0.1.0 — 大改版：引导流程 + 设置页 + 使用指南
- 新用户首次打开有 4 步引导（欢迎→配置→工作流→收藏）
- 行业配置从主界面搬到独立设置页
- 加了使用指南页，7 个功能逐条说明
- 镜像模式：提词器支持水平翻转，用于反射式提词器

## v0.1.1 — 提词器按钮太小
录视频的时候按 A-/A+/播放这些按钮太小了，手指戳不准。把 textSize 从 16 调到 20，padding 加大。

## v0.1.2 — API Key 处理
把明文硬编码的 API Key 改成 base64 编码。创建了 SecureStoragePlugin（Android EncryptedSharedPreferences），虽然最后决定 key 内置不让用户改，但加密存储的能力留着了。设置页去掉了 API Key 输入框。

## v0.1.3 — Release 签名
生成了 keystore，配了 release 签名。第一次打出正式签名的 APK，可以上架应用商店了。3.8MB。

## v0.1.4 — 问题数量跟时长走
之前不管选几分钟都生成 5-6 个问题。现在改成：1分钟=3个问题，3分钟=6个问题，5分钟=9个问题。问题也要求更简短，不超过 25 个字。

## v0.1.5 — UI 大调整
- 菜单 ☰ 从右上角移到左上角
- 顶部整体下移 12px，之前太挤了
- ⭐ 图标换成「收藏」文字
- 收藏后弹出气泡提示"收藏的内容在这里哦 👆"，指向左上角菜单，点任意位置关闭
- 生成结果加了「重新生成」按钮
- 选「问题」模式时，时长标签自动切换成问题数量（1分钟→3个问题），字号缩小防拥挤

## v0.1.6 — 提词器可拖动
第一次尝试在控制条上加拖动，失败了——按钮的 OnClickListener 吃掉了触摸事件。改成在提词器顶部加了专门的拖动手柄条（━━━━━━━━），按住就能拖整个窗口到任意位置。

## v0.1.7 — 隐私政策 + 上架准备
菜单加了「🔒 隐私政策」入口，写了完整的隐私政策页面（信息收集、网络请求、权限说明、数据安全、第三方服务、未成年人保护）。上架国内应用商店的技术准备基本完成。
